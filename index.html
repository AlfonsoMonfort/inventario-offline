      Instrucciones de uso

      1. Rellena fecha, almacén y vendedor.
      2. Pulsa empezar inventario.
      3. Selecciona cantidad (por defecto 1).
      4. Acerca el lector al código de barras.
      5. Pulsa la pantalla de escaneo.
      6. Separa poco a poco el móvil del lector.
      7. Si sale mensaje en verde = OK. Si es rojo = acerca y separa de nuevo.
      8. Finaliza para generar el Excel.
      9. Busca el Excel en el gestor de archivos / Files o similar.
